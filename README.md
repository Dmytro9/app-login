## How to use:

```
- clone the project locally
- add to api directory .env file with the jwt secret key (.env.example file is provided in 'api' dir))
- run docker-compose build --no-cache
- run docker-compose up
- navigate to http://localhost:3000
```

## Testing:
```
- make sure the node -v is ^10
- cd client 
- npm test
- cd api 
- npm test
```
